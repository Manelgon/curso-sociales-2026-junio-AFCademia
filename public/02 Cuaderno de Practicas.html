02Bloque 2 · Construcción

Tu hoja de ruta de los 13 nodos.

Marca cada nodo cuando esté configurado y guardado. No sirve si está a medias.

  • N1Trigger · Watch emails (Gmail/Outlook). Connection conectada y query newer_than:1d puesto.
  • N2IA · Chat Completion (clasificador). Prompt 1 pegado, temperature 0, response format JSON Object.
  • N3Parse JSON. Conectado al resultado del Nodo 2.
  • N4Router con 5 ramas. Filtros configurados (Contabilidad, Incidencias, Jurídico, Documentación, Otros como fallback).
  • N5Move email · Contabilidad → carpeta 1- 📊 Contabilidad.
  • N6Move email · Incidencias → carpeta 2- 🛠️ Incidencias.
  • N7Move email · Jurídico → carpeta 3- ⚖️ Jurídico.
  • N8Move email · Documentación → carpeta 4- 📄 Documentación.
  • N9Move email · Otros → carpeta 5- 📦 Otros.
  • N10Filter · ¿hay que responder? Condición: {{3.responder}} = true.
  • N11IA · Asunto. Prompt 2 pegado, max_tokens 70, temperature 0.4, response format Text.
  • N12IA · Cuerpo. Prompt 3 pegado, max_tokens 900, temperature 0.5, response format Text.
  • N13Create draft. To = remitente original, Subject = resultado N11, Content = resultado N12.
